# EIA Voice Platform

A beautiful, Instagram/Facebook-style anonymous school social platform built with Flask + SQLite.

## Features
- 👻 Anonymous posting — users can post without revealing their identity
- 🔍 Super Admin identity reveal — trace abuse when needed
- ❤️ Reactions (heart, fire, lightbulb) on all posts
- 💬 Comments with optional anonymity
- 👥 Follow system with follower feeds
- 💌 Private DMs (with anonymous mode)
- 🔔 Real-time notifications
- 🛡️ Admin panel with user management, flagging, role changes
- 📱 Fully responsive (mobile-friendly)
- 🎨 Beautiful dark design with Fraunces + Instrument Sans typography

## Roles
| Role | Description |
|------|-------------|
| `student` | Can post anonymously to whole school, comment, react, DM |
| `teacher` | Identified posting, can reach senate/admins |
| `senator` | Student representative, can post to senate channel |
| `admin` | Moderate content, view all channels |
| `super_admin` | Full access — can reveal anonymous identities |

## Local Development
```bash
pip install -r requirements.txt
python init_db.py
python app.py
# Open http://localhost:5000
```

## Deploy on Render
1. Push this folder to a GitHub repository
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your GitHub repo
4. Render will auto-detect `render.yaml` and configure everything
5. The persistent disk keeps your database and uploads across restarts

## Environment Variables
| Variable | Description |
|----------|-------------|
| `SECRET_KEY` | Flask session secret (auto-generated by Render) |
| `PORT` | Server port (set by Render automatically) |
